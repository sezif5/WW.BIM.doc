# Экспорт NWC

## Что делает команда

- Экспортирует выбранные модели Revit (`*.rvt`) в формат Navisworks (`*.nwc`).
- Сохраняет все `*.nwc` в одну выбранную папку **без создания подпапок** (один файл на каждую модель, с тем же именем).
- Для каждого файла использует 3D-вид `Navisworks`:
  - при необходимости создаёт его автоматически;
  - на виде скрывает аннотации, уровни, оси, связи и импортированные DWG/DXF;
  - отключает границы 3D-вида (section box).
- Открывает модели с рабочими наборами «все, кроме начинающихся с `00_`».
- По итогам выводит лог экспорта и время обработки каждого файла в окне pyRevit.

## Как пользоваться

1. Нажмите кнопку **«Экспорт NWC»** на панели pyRevit.
2. В первом диалоге выберите одну или несколько моделей Revit (`*.rvt`) для экспорта и нажмите **ОК**.
3. Во втором диалоге выберите папку, куда сохранить `*.nwc`:
   - если закрыть окно, не выбрав папку, файлы будут сохранены в `Документы\NWC_Export`.
4. Дождитесь завершения:
   - в окне pyRevit появится список обработанных моделей и путей к созданным `*.nwc`;
   - для каждой выбранной модели будет создан один `*.nwc` в выбранной папке.

## Важно

- Команда создаёт/использует 3D-вид `Navisworks` в каждой модели; вид остаётся в файле.
- Экспорт выполняется только по элементам, которые видимы на этом виде (аннотации, уровни, оси, DWG и связи скрываются).
- Скрипт рассчитан на работу в Revit 2022–2023.
