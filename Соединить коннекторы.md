# Инструкция по использованию команды «Выравнивание по коннекторам»

## Что делает команда

[Вывод] Команда выравнивает **второй выбранный инженерный элемент** по свободному коннектору **первого элемента** и пытается автоматически создать между ними соединение.

---

## Как пользоваться

[Вывод] 1. В Revit выделите **два инженерных элемента** с коннекторами (трубы, воздуховоды, оборудование и т.п.).  
[Вывод] • Первый выбранный элемент остаётся на месте.  
[Вывод] • Второй выбранный элемент будет перемещён.

[Вывод] 2. Нажмите кнопку скрипта на панели **pyRevit**.

[Вывод] 3. Если у элементов несколько свободных коннекторов, появится окно:  
[Вывод] • Выберите нужный коннектор для первого элемента.  
[Вывод] • Затем выберите коннектор для второго элемента.

[Вывод] 4. Скрипт автоматически:  
[Вывод] • Переместит второй элемент так, чтобы выбранные коннекторы совпали.  
[Вывод] • Попробует создать соединение между этими коннекторами.

---

## Важно помнить

[Вывод] • Команда работает только с элементами, у которых есть **свободные** коннекторы.  
[Вывод] • Если у элемента **один** свободный коннектор, он выбирается **автоматически**, без окна выбора.  
[Вывод] • Если подходящих коннекторов нет, скрипт покажет предупреждение и не будет ничего менять в модели.
