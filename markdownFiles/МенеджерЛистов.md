## Что делает команда

- Показывает список листов выбранного «альбома» (по параметру сортировки, по умолчанию `ADSK_Штамп Раздел проекта`).
- Позволяет менять порядок листов, копировать лист и редактировать прямо в таблице:
  - имя листа;
  - формат листа (параметр `A` у основной надписи).
- Автоматически нумерует листы выбранного альбома:
  - записывает номер в системный параметр «Номер листа»;
  - записывает числовой номер (без префикса) в выбранный параметр нумерации (по умолчанию `DVLK_Штамп_Номер листа`).

## Как пользоваться

1. Нажмите кнопку скрипта на панели **pyRevit** — откроется окно «Менеджер листов | Нумерация».
2. В верхней части окна:
   - выберите **«Параметр сортировки»** (обычно `ADSK_Штамп Раздел проекта`);
   - в поле **«Альбом»** выберите нужный альбом — в таблице отобразятся только листы этого альбома;
   - в **«Параметр нумерации»** выберите параметр для записи номера (по умолчанию `DVLK_Штамп_Номер листа`);
   - в поле **«Начинать с»** укажите стартовый номер.
3. В таблице:
   - стрелками **«▲ Вверх»** и **«▼ Вниз»** меняйте порядок листов;
   - кнопкой **«Копировать»** создайте копию выбранного листа сразу под ним;
   - для переименования листа дважды щёлкните по ячейке в колонке **«Имя листа»** и введите новое имя;
   - для изменения формата дважды щёлкните по ячейке в колонке формата `A`, введите число и нажмите Enter.
4. Нажмите **«Нумеровать»** — скрипт пронумерует листы по текущему порядку в таблице.
5. Нажмите **«Закрыть»** для выхода.

## Важно

- В менеджер попадают только обычные листы, без листов-заглушек.
- Нумерация выполняется только для листов выбранного альбома и только в текущем порядке.
- Формат `A` должен быть целым числом.
