# Экспорт RVT

## Что делает команда

- Открывает выбранные модели Revit (в том числе с **Revit Server / RSN://**) в фоне.
- Для каждой модели создаёт **отсоединённую копию RVT** в локальной папке.
- По умолчанию:
  - выполняет Detach с **сохранением рабочих наборов** (Detach & Preserve);
  - сохраняет файл как новый «центральный» в выбранной папке.
- Складывает все получившиеся `*.rvt` в **одну выбранную папку** (без подпапок), с тем же именем файла.
- В окно pyRevit выводит лог по каждому файлу: путь, результат, время открытия и сохранения.

## Как пользоваться

1. Нажмите кнопку **«Экспорт RVT»** на панели pyRevit.
2. В диалоге выбора файлов:
   - укажите одну или несколько моделей `*.rvt`
   - можно выбирать как локальные/сетевые файлы, так и пути Revit Server (RSN://).
3. В следующем окне выберите папку, куда сохранять копии:
   - если папка не выбрана, файлы будут сохранены в `Документы\RVT_Detached`
     (скрипт сообщит об этом в выводе).
4. Дождитесь завершения:
   - в окне pyRevit появится отчёт по всем моделям;
   - в указанной папке будут лежать новые отсоединённые файлы `*.rvt`.

## Важно

- Исходные модели **не изменяются**, создаются только новые файлы в целевой папке.
- При наличии файла с тем же именем он будет **перезаписан**.
- Для ускорения открытия используются настройки рабочих наборов, аналогичные экспортёру NWC.
