# Суперфильтр (3D / планы / разрезы)

## Что делает команда

- Фильтрует **элементы модели** по значениям параметров.
- Работает на:
  - текущем виде (только видимые элементы),
  - либо по всей модели.
- Позволяет:
  - задавать до **5 условий** по параметрам;
  - выбирать логику **«И»** или **«ИЛИ»** между условиями;
  - учитывать **параметры экземпляров, типов и (по кнопке) параметров семейств**.
- В результате **выделяет** найденные элементы в модели (с возможностью выбрать либо все, либо только 5 штук).

---

## Как пользоваться

1. **Активируйте обычный вид** (3D / план / разрез), не шаблон.
2. Нажмите кнопку **«Суперфильтр»** на панели pyRevit.
3. Вверху окна:
   - включите/выключите флажок **«Только видимые на активном виде»**;
   - при необходимости отключите или включите **«Включать параметры типов»**;
   - если нужны параметры самих семейств, нажмите **«Загрузить параметры семейства»**.
4. В блоке условий (строки «Условие 1…5»):
   - в первом выпадающем списке выберите **параметр** (можно печатать и выбирать из списка);
   - во втором — **операцию**: `=`, `!=`, `пусто`, `не пусто`;
   - в третьем (если не «пусто/не пусто») выберите или введите **значение**.
5. В поле **«Логика»** сверху справа выберите:
   - **«И»** — элемент должен удовлетворять всем условиям,
   - **«ИЛИ»** — достаточно одного выполняющегося условия.
6. Нажмите **«Посчитать»**, чтобы увидеть, сколько элементов подпадает под фильтр.
7. Для выбора элементов:
   - **«Выбрать 5 шт»** — выделит первые 5 найденных элементов;
   - **«Выбрать все»** — выделит все найденные элементы на виде/в модели.

Revit приблизит выбранные элементы на экране.

---

## Важно

- Фильтруются только **модельные элементы** (камеры игнорируются).
- Команда **ничего не изменяет** в параметрах и геометрии, она только ищет и выделяет элементы.
- Если по заданным условиям ничего не найдено, будет показано сообщение «Элементы не найдены по выбранным условиям».
