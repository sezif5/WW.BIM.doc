# Выбор элемента в связи и копирование ID

## Что делает команда

- Позволяет щёлкнуть по элементу **в связанном файле Revit (RVT-связь)** или в **активной модели**.
- Определяет **ID именно выбранного элемента** (в том числе внутри связи).
- Копирует этот ID в **буфер обмена Windows**.
- Выделяет и приближает:
  - для связи — сам экземпляр связи, внутри которой лежит элемент;
  - для активной модели — выбранный элемент.

---

## Как пользоваться

1. Нажмите кнопку скрипта на панели **pyRevit**.
2. В ответ на запрос в строке состояния:
   - кликните по элементу в **связанной модели** (RVT-связь),
   - либо по элементу в **текущей модели**.
3. После выбора:
   - появится окно с ID элемента;
   - ID будет скопирован в буфер обмена;
   - в модели автоматически выделится и приблизится соответствующий элемент
     (или связь, если элемент из RVT-ссылки).

---

## Важно

- Если документ связи выгружен или элемент недоступен, скрипт покажет сообщение об ошибке.
- Если буфер обмена занят другим приложением, копирование ID может не получиться — это будет указано в сообщении.
