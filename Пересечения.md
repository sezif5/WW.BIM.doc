# Пересечения (Navisworks XML → Revit)

## Что делает команда

- Открывает XML-отчёт Navisworks по коллизиям.
- Показывает список **всех тестов** (включая тесты без пересечений).
- Фильтрует строки так, чтобы остались только пересечения, относящиеся к **текущей модели Revit**.
- Для выбранных тестов выводит в окне pyRevit:
  - краткую сводку по количеству коллизий и парам категорий;
  - подробные таблицы со:
    - снимком коллизии;
    - названием проверки и именем пересечения;
    - ID элемента (кликабельно);
    - путями обоих элементов по структуре Navisworks с подсветкой категории.

## Как пользоваться

1. Нажмите кнопку **«Пересечения»** на панели pyRevit.
2. В диалоге выберите XML-отчёт Navisworks (`*.xml`).
3. В появившемся списке тестов:
   - отметьте нужные проверки (можно несколько, есть строка поиска);
   - нажмите **«Показать»**.
4. После обработки в окне pyRevit появится:
   - дата отчёта Navisworks;
   - сводка по выбранным проверкам;
   - ниже — таблицы по каждой проверке.

## Работа с отчётом

- Щёлкните по **ID** в таблице — соответствующий элемент выделится в модели Revit.
- Можно последовательно кликать по разным ID и быстро просматривать все коллизии.
- По скриншоту рядом удобно сопоставлять пересечение из Navisworks с элементами в модели.

## Важно

- В отчёт попадают только строки, где ID элемента существует в текущей модели Revit.
- Строки, относящиеся к другим файлам/моделям, по умолчанию отфильтрованы.
- Скрипт **ничего не изменяет** в модели, он только читает XML и строит отчёт.
