1. Разместите шаблонную типовую аннотацию "ТипАн_Мрк_В4Е_Уровень" на 3D-виде. Для этого:
	о Если семейства нет в проекте - скопируйте его из папки "Z:\02_Библиотека\01_Семейства\02_Сторонние\А101\ТипАн_Мрк_В4Е_Уровень.rfa"
	о Включите границы вида
	о Включите видимость типовых аннотаций в переопределении видимости и графики
	о Разместите загруженную аннотацию на любом 2D виде. Вырежьте ее и вставьте на целевой вид (Ctrl-X + Ctrl-v). Данное действие обусловлено невозможностью размещения таких элементов на 3D-видах по умолчанию.
	о Обратите внимание, галочка "Обрезать аннотации" может не позволять вам увидеть размещенный элемент.
2. При вызове команды требуется выделить любое число воздуховодов/трубопроводов. Если они были выделены заранее - выбирать не требуется
3. Выберите имена уровней, которые должны быть промаркированы.
4. Повторное применение без удаления старых марок вызовет задвоение.
5. Промаркировано будет пересечение элементов отметок уровней проекта, а не только тех уровней, которые физически затрагивают элемент.
